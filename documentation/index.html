⚡ Plugin Alpha ESS

Plugin A.V.A.T.A.R pour monitorer votre installation solaire et batterie Alpha ESS via l'Open API officielle.

📋 Prérequis

  1. Avoir un compte sur open.alphaess.com
  2. Enregistrer votre onduleur (numéro de série + CheckCode)
  3. Récupérer votre AppID et AppSecret

⚙️ Configuration

Éditez le fichier alphaEss.prop et renseignez vos identifiants :

{
  "modules": {
    "alphaEss": {
      "appId": "VOTRE_APP_ID",
      "appSecret": "VOTRE_APP_SECRET",
      "serialNumber": "VOTRE_NUMERO_SERIE"
    }
  }
}
⚠️ Ne partagez jamais votre AppSecret. Respectez un intervalle minimum de 10 secondes entre les appels API.

🎤 Commandes vocales

IntentionExemples de phrases
solarProduction"énergie produite aujourd'hui", "production solaire journée"
batteryLevel"niveau de la batterie", "charge batterie"
currentPower"puissance instantanée", "production maintenant"
fullStatus"état complet de l'installation", "bilan solaire"

🔄 Rafraîchissement automatique

Le plugin utilise un cron toutes les 10 minutes pour mettre à jour les données en arrière-plan. Les réponses vocales sont donc quasi-instantanées car les données sont déjà en cache.

Vous pouvez modifier la fréquence dans alphaEss.prop (champ cron.alphaEss.time). L'intervalle minimum recommandé par Alpha ESS est de 10 secondes.
